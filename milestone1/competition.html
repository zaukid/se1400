Us vs Them

Hasbro

Game Examples

  • Trivial Persuit
  • Scrabble
  • Monopoly

Pros

  • A lot of game titles
  • Excellent graphics
  • Easy to navigate
  • Provide links to make purchase
  • Allows for customer reviews
  • Extensive board game descriptions
  • Downloadable game instructions

Cons

  • They sell a lot more than board games
  • Large inventory makes site navigation take longer
  • Overhwelming edition and add-on options

Play Monster

Game Examples

  • 5 Second Rule
  • Quiddler
  • Spoons

Pros

  • A lot of game titles
  • Lots of games with fun custom pieces
  • Plenty of games for young children
  • Allows for customer reviews
  • Extensive board game descriptions

Cons

  • Buggy site with delays for auto-loading videos
  • They sell a lot more than board games
  • Online reviews available but few reviews left
  • Lost/broken custom pieces ruin the game
  • No provided links to purchase
  • No downloadable game instructions

Asmodee

Game Examples

  • Ticket to Ride
  • Pandemic
  • Catan

Pros

  • A solid number of outstanding game titles
  • Lots of games with fun custom pieces
  • Allows for customer reviews
  • Decent board game descriptions
  • Allows for direct purchasing
  • Easy to navigate site with extensive available filtering

Cons

  • Games for teens, young adults, and adults have a steep learning curve
  • Overhwelming edition and add-on options
  • Lost/broken custom pieces ruin the game
  • No downloadable game instructions
  • Games tend to be expensive and direct purchasing may not be cheapest option

Why Beyond the Board is better

  • Unique games unavailable elsewhere
  • Quickly find and purchase a fun game
  • Offer easy-to-learn games that are still fun for adults to play
  • Direct pricing is the same as buying elsewhere
  • Offer online reviews and a game feedback portal
  • Few custom pieces which can be easily replaced with affordable substitutes
  • Downloadable game instructions
